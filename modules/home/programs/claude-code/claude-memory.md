# Version Control

When working in a repository, check for a `.jj` directory. If present, use `jj` commands instead of `git` commands:
- Use `jj status` instead of `git status`
- Use `jj log` instead of `git log`
- Use `jj diff` instead of `git diff`
- Use `jj git push` instead of `git push`
- Use `jj git fetch` instead of `git fetch`
- Use the `/jujutsu` skill for commits and version control operations
